# 🂡 BlackJack for Beginners
A simple and beginner-friendly Blackjack game built with JavaScript to practice core programming concepts 
while creating a fun, interactive card game.

## 📸 Preview
![Preview](./assets/blackjack%20screenshot.png)

## Features
- 🎮 Start and restart the game anytime
- Drawing new cards
- Beginner-focused logic and structure

## 🛠️ Built With
- HTML
- CSS
- JavaScript
